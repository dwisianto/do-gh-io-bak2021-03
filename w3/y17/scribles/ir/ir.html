Multiple levels Normalized Discounted Cumulative Gain

Precision @K

1. Set a rank threshold K
2. Compute the number of relevant documents in top K
3. Ignores documents ranked lower than K
Example:
Suppose a search engine returns five documents. The relevance documents are located at position 1,3, and 5, while the documents at position 2 and 4 are not relevant
Prec@1=1/1
Prec@2=1/2
Prec@3=2/3
Prec@4=2/4
Prec@5=3/5

Average precision - Mean Average Precision

Definition Average Precision (AP_q) is the average of Precision@K.
Mean Average Precision (MAP) even out the Average Precision (AP_q) across multiple queries.
Algorithm Compute Precision@K for all R relevant documents for a specific query.
- K_1 is the rank position for the first relevant document.
- K_2 is the rank position for the second relevant document.
- K_R is the rank position for the R^{th} relevant document.
- AP = (K_1+K_2+K_R)/R
Query1 AP= 1/3 (1/1+2/3+3/5)
Prec@1=1/1
Prec@2=1/2
Prec@3=2/3
Prec@4=2/4
Prec@5=3/5
Query2 AP= 1/2 (1/2+2/4)
Prec@1=0/1
Prec@2=1/2
Prec@3=1/3
Prec@4=2/4
Prec@5=2/5
Mean Average Precision MAP = 1/2( AP_1 + AP_2 )

Normalized Discounted Cumulative Gain Normalized Discounted Cumulative Gain (NDCG) is a method for measuring the quality of a set of search results . It asserts the following: Very relevant results are more useful than somewhat relevant results which are more useful than irrelevant results (cumulative gain). Relevant results are more useful when they appear earlier in the set of results (discounting). The result of the ranking should be irrelevant of the query preformed (normalization).

Normalized Discounted Cumulative Gain (NDCG) is a method for measuring the quality of a set of search results . It asserts the following:

  • Very relevant results are more useful than somewhat relevant results which are more useful than irrelevant results (cumulative gain).
  • Relevant results are more useful when they appear earlier in the set of results (discounting).
  • The result of the ranking should be irrelevant of the query performed (normalization).

DCG originates from an earlier measure called Cumulative Gain. $$CG_p = \sum_{i=1}^p rel_i$$
